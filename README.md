# MPC e-Tender

## Project tracking

- **`BUGS.md`** — open bugs, tech debt, and per-entry detail. Engineer-owned (Claude Code writes via fix commits).
- **`docs/PM_BACKLOG.md`** — priority queue, scope decisions pending, recently shipped log. PM-owned (Claude drafts via prompts; Claude Code commits).
- **`docs/prompts/`** — archive of every Claude Code prompt issued for this project.
- **`docs/CHANGELOG.md`** — shipped releases.

When opening a fresh Claude Code session for this repo, the convention is to read `BUGS.md` and `docs/PM_BACKLOG.md` before doing other work — the queue state lives there.
